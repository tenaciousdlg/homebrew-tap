# homebrew-tap

Personal Homebrew tap for [tenaciousdlg](https://github.com/tenaciousdlg)'s tools.

```sh
brew install tenaciousdlg/tap/teleport-audit-report
```

Formulas here are published automatically by [goreleaser](https://goreleaser.com)
from each tool's own repo on release — don't edit them by hand.
